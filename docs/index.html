🚀 クイックスタート

5分以内にアジャイル環境をセットアップ:

# リポジトリをクローン
git clone https://github.com/pandalize/agile.git
cd agile

# セットアップスクリプトを実行
./scripts/setup-enhanced.sh -r your-org/your-repo -s "Sprint 1" -d 14

# ダッシュボードを表示
open https://your-org.github.io/your-repo/
